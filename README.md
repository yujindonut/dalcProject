## How to Use the Project

> 1. Start 버튼 클릭
> 2. 자신의 사진 업로드
> 3. 자신과 닮은 멸종위기 동물에 대한 정보를 본다
> 4. Yes -> 다양한 뉴스
> 5. No -> 다양한 기부사이트
